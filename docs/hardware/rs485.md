# RS485 Interface

## Overview

stepSOLAR uses UART2 of the ESP32-S3 for RS485 Modbus RTU communication. A half-duplex RS485 transceiver (MAX485 or equivalent) handles bus direction switching via the `DE/RE` pin.

## Wiring

```
ESP32-S3          MAX485 / SP3485         RS485 Bus
GPIO15 (TX) ───── DI
GPIO16 (RX) ───── RO
GPIO14 (DIR)───── DE ─┐
               RE  ┘              A (+) ────── to master
3.3V ──────────── VCC             B (−) ────── to master
GND ─────────────  GND            GND   ────── (optional shield)
```

!!! note "DE and RE tied together"
    Both DE (Driver Enable) and RE (Receiver Enable) are connected to GPIO14. HIGH enables transmit, LOW enables receive. This is standard half-duplex RS485 operation.

## Configuration

All RS485 parameters are configurable via the **Settings** tab and stored in EEPROM:

| Parameter | Default | Range |
|-----------|---------|-------|
| Baud rate | 9600 | 1200, 2400, 4800, 9600, 19200, 38400, 57600, 115200 |
| Parity | None | None, Even, Odd |
| Stop bits | 1 | 1, 2 |
| Slave ID | 12 | 1–247 |

Changes take effect immediately — the UART is restarted without rebooting the device.

## Supported UART Configurations

| Parity | Stop bits | UART Config |
|--------|-----------|-------------|
| None | 1 | `SERIAL_8N1` (default) |
| None | 2 | `SERIAL_8N2` |
| Even | 1 | `SERIAL_8E1` |
| Even | 2 | `SERIAL_8E2` |
| Odd | 1 | `SERIAL_8O1` |
| Odd | 2 | `SERIAL_8O2` |

## Bus Termination

For cable runs longer than ~10 m, add a 120 Ω termination resistor between A and B at both ends of the bus.
