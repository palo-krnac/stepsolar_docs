# RS485 Settings

Modbus RS485 parameters are fully configurable via the **Settings** tab and stored in EEPROM. Changes apply immediately without rebooting.

## Default Settings

| Parameter | Default |
|-----------|---------|
| Baud rate | 9600 |
| Data bits | 8 (fixed) |
| Parity | None |
| Stop bits | 1 |
| Slave ID | 12 |

## Available Baud Rates

`1200` · `2400` · `4800` · `9600` · `19200` · `38400` · `57600` · `115200`

## Available Configurations

| Parity | Stop bits | Common use |
|--------|-----------|-----------|
| None | 1 | Most PLCs, energy meters |
| None | 2 | Some older equipment |
| Even | 1 | Siemens, Schneider |
| Odd | 1 | Some Mitsubishi devices |

!!! warning "Match settings on both ends"
    The Modbus master (PLC, SCADA, Home Assistant) must use **identical** baud rate, parity and stop bits. A mismatch causes garbled frames and no communication.
